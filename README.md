# Student Management System

A complete full-stack student management application built with Next.js, SQLite, and NextAuth.

## Features

- **Student CRUD Operations**: Create, read, update, and delete student records
- **Authentication**: Secure login system using NextAuth with email and password
- **Database**: SQLite for persistent data storage
- **Responsive UI**: Clean and modern interface with Tailwind CSS
- **Full-Stack**: Backend API and frontend all in one Next.js application

## Quick Start

1. Clone and install:
```bash
git clone https://github.com/absqallihoussaini-ui/sm.git
cd student-management
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Open http://localhost:3000 and login with:
   - Email: `admin@example.com`
   - Password: `admin123`
